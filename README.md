# Javascript Array Manipulation
This project containes javascript activites focused on Array Manipulaiton, searching, filtering, and rendering content in the browser

## Project Overview
The main idea of this project was to build and manage a simple shopping list using javascript.

### Task 1: Array Manipulation Basics
In this task, I created an empty array called 'shoppingList' and built function to manage it. The 'addItem' function adds a new item to the array. 'removeLastItem' removes the last item and display list

### Task 2: Filter and Search an Array
in the second task, I udpated the 'addItem' function so that it only adds an item if it is already not present in the array. I also created 'filterItems' function that takes a search term and returns matching items from the shopping list

### Task 3: Render the List in the Browser
In the final task I created an HTML page with an input field and 'Add Item' button and a Remove Last Item button and an unordered list to display the items.
